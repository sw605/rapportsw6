This project was developed as part of the multi-group project \gls{astep} in the spring of 2016.
The resulting system consists of three parts; an app, a server and an \gls{astep} intergration.

Throughout the project, we gained a lot of experience working with Android, Java Sockets, Http Requests, and working on large projects.
Which we hope can be seen throughout the report.

We would like to thank Davide, our supervisor, for his work and support throughout the project.
